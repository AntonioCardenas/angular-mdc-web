Drawers

Import

Design & API Documentation

Selector: mdc-drawer

Properties Description
drawer: string Set the drawer implementation.

Valid values: 'permanent' | 'dismissible' | 'modal'
fixedAdjustElement: any Optional. Reference to a drawer's attached body element.
Methods Description
open() Opens the drawer.
close() Closes the drawer.
isOpen(): boolean Returns whether or not the drawer is open.
Events Description
opened() Event dispatched on drawer open.
closed() Event dispatched on drawer close.

Selector: mdc-drawer-header

Optional. Non-scrollable element that exists at the top of the drawer.
title: string Optional. Title text element of the drawer.
subtitle: string Optional. Subtitle text element of the drawer.

Selector: mdcDrawerTitle

Optional. Title text element of the drawer.

Selector: mdcDrawerSubtitle

Optional. Subtitle text element of the drawer.

Selector: mdc-drawer-content

Scrollable content area of the drawer.
