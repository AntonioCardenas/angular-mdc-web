Design & API Documentation

mdc-tab-bar

Properties Description
stacked: boolean Indicates that the tab icon and label should flow vertically instead of horizontally.
fixed: boolean Indicates that the tab should shrink in size to be as narrow as possible without causing text to wrap.
fade: boolean Sets up the tab indicator to fade in on activation and fade out on deactivation. Default is false.
iconIndicator: string Denotes an icon to use as a tab indicator.
align: string Sets the elements inside the scroll content element to be aligned to the start, center or end of the scroll content element. 
Valid values: 'start' | 'center' | 'end'
* Only for Fixed Tabs.
Methods Description
activateTab(index: number) Activates the Tab at the given index.
scrollIntoView(index: number) Scrolls the Tab at the given index into view.
getActiveTabIndex(): number Returns the index of the active Tab.
getActiveTab(): MdcTab | undefined Returns the MDCTab that is active.
Events Description
activated: (source: MdcTabBar, index: number, tab: MdcTab) Emitted when a tab is activated.

mdc-tab-scroller

Methods Description
scrollTo(scrollX: number) Scrolls to the scrollX value.
incrementScroll(scrollXIncrement: number) Increments the current scroll value by the scrollX value.
getScrollPosition(): number Returns the current visual scroll position.

mdc-tab

Properties Description
label: string Optional. Adds a label in the tab.
icon: string Optional. Indicates a leading icon in the tab.

mdc-tab-label, mdcTabLabel

Optional. Allows customizing a label in the tab.

mdc-tab-icon, mdcTabIcon

Optional. Allows customizing a leading icon in the tab.
