Sass Mixins

To customize an icon button's color and properties, you can use the following mixins.

Mixin Description
icon-button-size($width, $height, $padding) Sets the width, height, font-size and padding for the icon and ripple. $height is optional and defaults to $width. $padding is optional and defaults to max($width, $height)/2. font-size is set to max($width, $height).
icon-button-ink-color($color) Sets the font color and the ripple color to the provided color value.
