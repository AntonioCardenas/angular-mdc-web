MdcDrawer

Selector: mdc-drawer

Exported as: mdcDrawer

Properties

Name Description
drawer: string Set the drawer implementation.

Valid values: dismissible | modal | empty
open: boolean Opens or closes the drawer.
fixedAdjustElement: any Optional. Reference to a drawer's attached body element.
dir: string Optional. If value is 'rtl', drawer opens from the right side of the screen.
autoFocus: boolean Whether the drawer should focus the first focusable element on open. (Default: true)
restoreFocus: boolean Whether the drawer should restore focus to the previously-focused element, after it's closed. (Default: true)

Events

opened() Event dispatched on drawer open.
closed() Event dispatched on drawer close.
openedChange(opened: boolean) Event dispatched on drawer open state changes.
drawerChange() Event dispatched when the drawer variant changes.

Focus Management

By default, the first element within the drawer will receive focus upon open. This can be configured by setting the cdkFocusInitial attribute on another focusable element or setting autoFocus to false.
