MdcSelect

Selector: mdc-select

Exported as: mdcSelect

Properties

Name Description
id: string Unique Id of the select (auto generated if not supplied).
name: string Name of the select.
placeholder: string Text shown if no value has been selected.
value Sets the selected item by value.
tabIndex: number Set the underlying tab index of the select. (Default is 0)
floatLabel: boolean Whether or not to show or hide a floating placeholder.
outlined: boolean Styles the select as an outlined select.
disabled: boolean Enables/disables the select.
helperText: MdcHelperText Reference to related MdcHelperText
compareWith: (o1: any, o2: any) => boolean Function to compare the option values with the selected values. The first argument is a value from an option. The second is a value from the selection. A boolean should be returned.

Methods

getSelectedIndex(): number Returns the index of the currently selected option. Returns -1 if no option is currently selected.
setSelectionByValue(value) Set selection to the passed value.
focus() Set focus to the select.

Events

valueChange(index: number, value: any) Event emitted on any value change.
selectionChange(source: MdcSelect, index: number, value: any) Event emitted if user changed the value.
blur(value) Emitted whenever the select loses focus.
focus(boolean) Emitted when the select gains or loses focus.

MDCSelectHelperText

Selector: mdc-select-helper-text mdcSelectHelperText

Exported as: mdcSelectHelperText

Properties

Name Description
validation: boolean Help text can be used to provide additional validation messages.
persistent: boolean Help text will always be visible.

Methods

showToScreenReader() Makes the helper text visible to the screen reader.
setValidity(inputIsValid: boolean) Sets the validity of the helper text.

MDC_SELECT_DEFAULT_OPTIONS

Injection token that can be used to configure the default options for all select's within an app.
