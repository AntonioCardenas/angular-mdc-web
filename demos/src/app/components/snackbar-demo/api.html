Service

MdcSnackbar

Service to open MDC snackbars.

Properties

Name Description
afterDismiss: Observable Gets an observable that is notified when the snackbar is finished closing.

Methods

show(message: string) Show snackbar with message.
show(message: string, action: string) Show snackbar message and action.
show(message: string, action: string, config?: MdcSnackbarConfig) Show the snackbar with optional configuration.

MdcSnackbarConfig

timeout: number (Default 2750) The amount of time in milliseconds to show the snackbar.
actionHandler: Function The function to execute when the action is clicked.
multiline: boolean Whether to show the snackbar with space for multiple lines of text.
actionOnBottom: boolean Whether to show the action below the multiple lines of text.
align: string Alignment of the snackbar on screen. Valid values - 'start' | 'center' | 'end'
dismissOnAction: boolean Dismisses snackbar when action button is clicked.
focusAction: boolean Sets focus to the action button.
