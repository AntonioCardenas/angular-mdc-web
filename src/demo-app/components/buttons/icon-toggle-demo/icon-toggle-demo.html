Icon Toggle Buttons

MDC Icon Toggle provides a Material Design icon toggle button. It is fully accessible, and is designed to work with any icon set.

References

  • Material Design guidelines: Toggle Buttons
  • Material Components Web

Import

Design & API Documentation

mdc-icon-toggle

Properties Description
on: boolean Returns whether or not the icon toggle is currently toggled on. Setting this property will update the toggle state.
iconClass: string Set selector to other icon libraries such as Font Awesome. (default is 'material-icon')
iconOn: string Set icon as content when toggled on. Use for Material Icons.
iconOff: string Set icon as content when toggled off. Use for Material Icons.
labelOn: string Set "aria-label" attribute when toggled on.
labelOff: string Set "aria-label" attribute when toggled off.
cssClassOn: string A css class to apply to the icon element when toggled on. Use for Font Awesome library.
cssClassOff: string A css class to apply to the icon element when toggled off. Use for Font Awesome library.
primary: boolean Colors the button with the primary theme color.
secondary: boolean Colors the button with the secondary theme color.
disabled: boolean Disables the toggle button.
Methods Parameters
toggle(isOn?: boolean) Toggles the button state. If an argument is given, will toggle on if true, off if false.
refreshToggleData() Allow's developers to refresh toggle data if needed.
isOn() Returns true if the button is toggled on, false otherwise.
Events Description
Broadcast when a user toggles the button.
