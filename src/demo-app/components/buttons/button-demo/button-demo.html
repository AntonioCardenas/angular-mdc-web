Design & API Documentation

button[mdc-button]

a[mdc-button]

Properties Description
primary: boolean Colors the button with the primary theme color.
secondary: boolean Colors the button with the secondary theme color.
dense: boolean Compresses the button text to make it slightly smaller.
raised: boolean A contained button that is elevated upon the surface.
unelevated: boolean A contained button that is flush with the surface.
outlined: boolean A contained button that is flush with the surface and has a visible border.
icon: any Icon to render within root element. If set, adds `mdc-button__icon` class to element.
disabled: boolean Disables the button.
Methods Description
setPrimary(primary: boolean) Colors the button with the primary theme color.
setSecondary(secondary: boolean) Colors the button with the secondary theme color.
setDense(dense: boolean) Compresses the button text to make it slightly smaller.
setRaised(raised: boolean) A contained button that is elevated upon the surface.
setUnelevated(unelevated: boolean) A contained button that is flush with the surface.
setOutlined(outlined: boolean) A contained button that is flush with the surface and has a visible border.
setIcon(icon: any) Icon to render within root element. If set, adds `mdc-button__icon` class to element.
setDisabled(disabled: boolean) Disables the button.
focus() Focuses the button.
