Buttons

Button is a spec-aligned button component adhering to the Material Design button requirements.
Component
button[mdc-button]
a[mdc-button]
Properties Description
primary: boolean Colors the button with the primary theme color.
secondary: boolean Colors the button with the secondary theme color.
dense: boolean Compresses the button text to make it slightly smaller.
raised: boolean Elevates the button and creates a colored background.
unelevated: boolean A button that is flush with the surface.
stroked: boolean A button that is flush with the surface and has a visible border.
icon: boolean Sets styling of icon element.
disabled: boolean Disables the button.
disableRipple: boolean 
Disable the ink ripple.
Example: disableRipple="true" or disableRipple
Methods Description
focus() Focuses the button.
