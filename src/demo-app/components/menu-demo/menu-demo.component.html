Menu

The MDC Menu component is a spec-aligned menu component adhering to the Material Design menu requirements.

API reference

Selectors
mdc-menu
Properties Description
@Input() openFrom: string Override the opening point of the menu. (default is TOP_LEFT) 
Valid values:
TOP_LEFT - Open the menu from the top left.
TOP_RIGHT - Open the menu from the top right.
BOTTOM_LEFT - Open the menu from the bottom left.
BOTTOM_RIGHT - Open the menu from the bottom right.
Methods Parameters
open(focusIndex: number) Opens the menu. Takes an focusIndex property that specifies the index of the menu item to be focused. If the focusIndex is omitted, no menu item will be focused.
isOpen(): boolean Returns whether or not the menu is open.
close() Closes the menu.
Event Description
cancel Dispatches an event that the menu closed with no selection made.
select Dispatches an event that a menu item has been selected.
Directives Description
mdc-menu-anchor
mdc-menu-item Single list item inside of a menu.
mdc-menu-divider Draws a line at the bottom of the menu item.
