Menus

The MDC Menu component is a spec-aligned menu component adhering to the Material Design menu specification.

Import

Design & API Documentation

mdc-menu

Properties Description
id: string Unique Id of the menu (auto generated if not supplied).
anchorCorner: string Override the opening point of the menu. (default: top-start) 
Values: 'top-start' | 'top-end' | 'bottom-start' | 'bottom-end'
rememberSelection: boolean Whether or not previous selection is focused upon menu open. (Default is false)
quickOpen: boolean Sets whether the menu should open and close without animation when the open/close methods are called.
anchor: MdcMenuAnchor Parent element to automatically anchor to when opened.
Methods Parameters
open() Opens the menu.
isOpen(): boolean Returns whether or not the menu is open.
close() Closes the menu.
toggle() Open or close the menu.
focus() Set focus to the menu element.
setAnchorCorner(corner: string) Override the opening point of the menu. 
Values: 'top-start' | 'top-end' | 'bottom-start' | 'bottom-end'
setQuickOpen(quickOpen: boolean) Sets whether the menu should open and close without animation when the open/close methods are called.
getSelectedIndex(): number Returns the selected item index.
setSelectedIndex(index: number) Index of the item to set as selected.
setMaxHeight(height: string) Sets max-height style for the menu element.
setRememberSelection( rememberSelection: boolean) Remembers previous selection, and focuses.
Event Description
cancel() Dispatches an event that the menu closed with no selection made.
select(item: MdcMenuItem, index: number) Dispatches an event that a menu item has been selected.

mdc-menu-item

Properties Description
id: string The unique ID of the option.
label: string The displayed label of the option.
disabled: boolean Disable selection of the select item.

mdc-menu-anchor

Use to anchor a menu to an element.

mdc-menu-divider

Draws a line at the bottom of the menu item.
