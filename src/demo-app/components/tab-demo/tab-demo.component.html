Tabs

Tabs enable content organization at a high level, such as switching between views, data sets, or functional aspects of an app.
Directive
mdc-tab-bar The main component which is composed of mdc-tab elements.
Properties Description
@Input() primaryColor: boolean Sets indicator bar line to use app primary color.
@Input() accentColor: boolean Sets indicator bar line to use app accent color.
Events Description
Broadcast when a user actions on a tab, resulting in a change in the selected tab.
Directive
mdc-tab The individual tab elements.
@Input() active: boolean Whether or not the tab is active. Setting this makes the tab active.
@Input() preventsDefaultOnClick: boolean Whether or not the tab will call preventDefault() on an event. Setting this makes the tab call preventDefault() on events.
@Input() disableRipple: boolean
Events Description 
]]>
 Broadcast when a user actions on the tab.
Child Directives Description
mdc-tab-icon Add icon to the tab.
mdc-tab-icon-text Text and icon are centered horizontally in the tab.
Directive
mdc-tab-bar-scroller The component which controls the horizontal scrolling behavior of an mdc-tab-bar that overflows its container.
Child Directives Description
mdc-tab-bar-scroll-frame The scroll frame is the parent element of the tab bar, and serves to mask the tabs in the tab bar when they overflow the available width.
mdc-tab-bar-scroll-back Scrolls the tab bar such that the leftmost tab traverses the scroll frame.
mdc-tab-bar-scroll-forward Scrolls the tab bar such that the rightmost tab traverses the scroll frame.
mdc-tab-bar-scroll-button Use to add a button to either [mdc-tab-bar-scroll-back] or [mdc-tab-bar-scroll-forward] directives.
