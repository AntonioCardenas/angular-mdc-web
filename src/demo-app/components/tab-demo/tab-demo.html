Tabs

Tabs enable content organization at a high level, such as switching between views, data sets, or functional aspects of an app.
Directive
mdc-tab-bar The main component which is composed of mdc-tab elements.
Properties Description
active: boolean Sets a tab as active.
primary: boolean Sets tab to use primary theme color.
secondary: boolean Sets tab to use secondary theme color.
selectedIndex: number Set the index of the active tab.
disableRipple: boolean 
Disable the ink ripple.
Example: disableRipple="true" or disableRipple
Methods Description
getNumberOfTabs(): number Returns the number of tabs.
getComputedWidth(): number Returns the width of the element containing the tabs.
layout() Set or refresh layout for the Tab Bar component.
getActiveTabIndex(): number Returns index of currently active tab.
switchToTabAtIndex(index: number, shouldNotify: boolean = true) Updates the active tab to be the tab at the given index, emits change event if shouldNotify is true.
setPreventDefaultOnClickForTabAtIndex(index: number, preventDefaultOnClick: boolean) Sets preventDefault to true or false on a passed tab index.
isDefaultPreventedOnClickForTabAtIndex(index: number): boolean Returns whether or not preventDefault is set on a tab.
setDisableRipple(disabled: boolean) Disable or enable ink ripples on tabs.
setActiveTab(tab: MdcTab, active: boolean) Use to enable or disable an active tab.
getActiveTab(): MdcTab Returns the active tab.
Events Description 
selectedTabChange: EventEmitter]]>
 Broadcast when a user actions on a tab, resulting in a change in the selected tab.
Directive 
mdc-tab
a[mdc-tab]
 The individual tab elements.
disabled: boolean Whether or not the tab is disabled. Setting this makes the tab disabled.
Child Directives Description
mdc-tab-icon-text Text and icon are centered horizontally in the tab.
Methods Description
getComputedWidth(): number Return the computed width for tab.
getComputedLeft(): number Return the computed left offset for tab.
isActive() Return true if tab is active.
setActive(boolean) Set tab to active.
getPreventDefaultOnClick(): boolean Return true if the tab prevents the default click action.
setPreventDefaultOnClick(boolean) Sets tabs preventDefaultOnClick property to the value of the preventDefaultOnClick argument passed.
measureSelf() Sets computedWidth and computedLeft for a tab.
setDisabled(isDisabled: boolean) Whether or not the tab is disabled.
Events Description
select(tab: TabComponent) Broadcast when a user actions on the tab.
Directive
mdc-tab-bar-scroller The component which controls the horizontal scrolling behavior of an mdc-tab-bar that overflows its container.
Child Directives Description
mdc-tab-bar-scroll-frame The scroll frame is the parent element of the tab bar, and serves to mask the tabs in the tab bar when they overflow the available width.
mdc-tab-bar-scroll-back Scrolls the tab bar such that the leftmost tab traverses the scroll frame.
mdc-tab-bar-scroll-forward Scrolls the tab bar such that the rightmost tab traverses the scroll frame.
Methods Description
scrollToTabAtIndex(index: number) Scrolls the tab bar such that the tab at the index provided traverses the scroll frame and becomes the leftmost tab.
layout() If the tab bar is overflowing its available width, this method will reset the back and forward indicators to the correct states (visible/hidden) based on the new width.
scrollBack(event?: Event) Scrolls the tab bar such that the leftmost tab traverses the scroll frame and becomes the rightmost tab.
scrollForward(event?: Event) Scrolls the tab bar such that the rightmost tab traverses the scroll frame and becomes the leftmost tab.
