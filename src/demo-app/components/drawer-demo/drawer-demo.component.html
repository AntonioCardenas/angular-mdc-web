Drawers

The MDC Drawer component is a spec-aligned drawer component adhering to the Material Design navigation drawer pattern. It implements permanent, persistent, and temporary drawers.
Component
mdc-permanent-drawer
Child Directives Description
mdc-permanent-drawer-spacer Optional. Add to node to provide the matching amount of space for toolbar.
mdc-permanent-drawer-content Mandatory. Needs to be set on the container node for the drawer content.
mdc-permanent-drawer-selected Optional. Highlights currently selected drawer item.
Component
mdc-persistent-drawer
Child Directives Description
mdc-persistent-drawer-spacer Optional. Add to provide the matching amount of space for toolbar.
mdc-persistent-drawer-header Optional. Add to create a 16:9 drawer header.
mdc-persistent-drawer-header-content Optional. Add content inside mdc-persistent-drawer-header.
mdc-persistent-drawer-content Mandatory. Needs to be set as the container node for the drawer content.
mdc-persistent-drawer-selected Optional. Highlights currently selected drawer item.
Events Description
@Output() opened Event dispatched on drawer open.
@Output() closed Event dispatched on drawer close.
Component
mdc-temporary-drawer
Child Directives Description
mdc-temporary-drawer-spacer Optional. Add to provide the matching amount of space for toolbar.
mdc-temporary-drawer-header Optional. Add to create a 16:9 drawer header.
mdc-temporary-drawer-header-content Optional. Add content inside mdc-temporary-drawer-header.
mdc-temporary-drawer-content Mandatory. Use for the drawer content.
mdc-temporary-drawer-selected Optional. Highlights currently selected drawer item.
Events Description
@Output() opened Event dispatched on drawer open.
@Output() closed Event dispatched on drawer close.
