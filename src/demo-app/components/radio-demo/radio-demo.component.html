Radio Button

The MDC Radio Button component provides a radio button adhering to the Material Design Specification.
Selectors
mdc-radio
Properties Description
@Input() id: string Unique Id of the radio button (auto generated if not supplied).
@Input() name: string Name of the radio button group.
@Input() value: any Value of the radio button.
@Input() ariaLabel: string Used to set the 'aria-label' attribute on the underlying input element.
@Input() ariaLabelledby: string The 'aria-labelledby' attribute takes precedence as the element's text alternative.
@Input() checked: boolean Use to verify the checked value.
@Input() disabled: boolean Disables the component.
Event Description
@Output() change Event dispatched on value change.
