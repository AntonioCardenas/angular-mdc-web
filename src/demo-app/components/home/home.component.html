Angular Material Design Components View on - GitHub

Buttons

 Buttons communicate the action that will occur when the user touches them.

Card

 A card is a sheet of material that serves as an entry point to more detailed information.

Checkbox

 Checkboxes allow the user to select multiple options from a set.

Drawer

 Drawers slide in from the left and contains the navigation destinations for your app.

Elevation

 Elevation is the relative depth, or distance, between two surfaces along the z-axis.

Floating Action Button (FAB)

 A floating action button represents the primary action in an application.

Linear Progress

 Progress and activity indicators are visual indications of an app loading content.

List

 Lists present multiple line items vertically as a single continuous element.

Menu

 Menus display a list of choices on a transient sheet of material.

Radio Button

 Radio buttons allow the user to select one option from a set.

Snackbar

 Snackbars provide brief feedback through a message at the bottom of the screen.

Switch

 On/off switches toggle the state of a single settings option.

Textfield

 Text fields allow users to input, edit, and select text.

Toolbar

 Toolbars appear a step above the sheet of material affected by their actions.

Typography

 Material Design typography uses the Roboto font.
