App Bar

MDC App Bar acts as a container for items such as application title, navigation icon, and action items.

Import

Design & API Documentation

mdc-app-bar

Properties Description
fixed: boolean Style the app bar as a fixed top app bar.
short: boolean Style the app bar as a short app bar. Short top app bars should only be used with one action item.
prominent: boolean Enables prominent variant.
dense: boolean Enables dense variant.
shortCollapsed: boolean Set the short app bar to always collapsed.
fixedAdjustElement: any If set, scroll event listener is attached to element object.
Methods Description
setFixed(fixed: boolean) Style the app bar as a fixed top app bar.
setShort(short: boolean) Style the app bar as a short app bar. Short top app bars should only be used with one action item.
setProminent(prominent: boolean) Enables prominent variant.
setDense(dense: boolean) Enables dense variant.
setShortCollapsed(shortCollapsed: boolean) Set the short app bar to always collapsed.
setFixedAdjustElement(element: HTMLElement) Set automatic adjustment for padding on element positioned below app bar.
Events Description
navigationSelected: MdcAppBarNavSelected Emitted when the app bar navigation menu icon is selected.

mdc-app-bar-row, [mdcAppBarRow]

A one-line row to add app bar sections.

mdc-app-bar-title

Styles toolbar text representing a page's title, or an application name.

mdc-app-bar-section, [mdcAppBarSection]

A content area for navigation or actions.
Properties Description
align: string Align section to either 'start' or 'end' of row. 
Valid values are 'start' | 'end'

mdc-app-bar-navigation-icon, mdc-icon[mdcAppBarNavIcon]

Left most icon of app bar used for navigation.

mdc-app-bar-action-item, mdc-icon[mdcAppBarActionItem]

Top app bars can accommodate multiple action items on opposite side of navigation icon.
