App Bar

MDC App Bar acts as a container for items such as application title, navigation icon, and action items. Top app bars scroll with content by default.

References

  • Material Design guidelines: Toolbar
  • Material Components Web

Import

Design & API Documentation

mdc-app-bar

Properties Description
short: boolean 
Style the app bar as a short app bar.
  Short top app bars should only be used with one action item.
prominent: boolean Enables prominent variant.
collapsed: boolean Indicate the short app bar is collapsed. (requires short is set to true)
fixedAdjustElement: any If set, scroll event listener is attached to element object.
Events Description 
navigationSelected:
  EventEmitter]]>
 Emitted when the app bar navigation menu icon is selected.

mdc-app-bar-row

A one-line row to add app bar sections.

mdc-app-bar-title

Styles toolbar text representing a page's title, or an application name.

mdc-app-bar-section

A content area for navigation or actions.
Properties Description
align: string Align section to either 'start' or 'end' of row. 
Valid values are 'start' | 'end'

mdc-icon[mdcAppBarNavIcon]

mdc-app-bar-navigation-icon

Left most icon of app bar used for navigation.

mdc-icon[mdcAppBarActionItem]

mdc-app-bar-action-item

Top app bars can accommodate multiple action items on opposite side of navigation icon.
