Component
mdc-text-field-box
Properties Description
id: string Unique id of the element.
type: string = 'text' Input type of textfield (e.g. email, password, url).
name: string Name of the textfield.
value: string The input element's value.
dense: boolean Shrinks the font size and height of the input.
disabled: boolean Disables the component.
required: boolean Whether the element is required.
label: string Shown to the user when there's no focus or values.
tabIndex: number Tab index of the text element.
maxlength: number Maxlength of characters allowed to be entered.
focused: boolean Set or unset focus on the input element.
direction: string Support for right-to-left (RTL). (Default is ltr) 
Values: 'ltr' | 'rtl'
Directive
mdc-text-field-helper-text Useful for providing supplemental information to users, as well for validation messages.
Properties Description
validation: boolean Help text can be used to provide additional validation messages.
persistent: boolean Help text will always be visible.
Leading/Trailing icons
leading Add a leading icon to the textfield. 
 Example: person]]>
trailing Add a trailing icon to the textfield. 
 Example: person]]>
Methods Description
setValid(value?: boolean) Updates input validity using passed value, or if left empty checking the validity of the input.
isBadInput(): boolean Returns input bad input validity value.
isDisabled(): boolean Returns whether or not the input is disabled.
focus() Sets focus to the input.
updateIconState() Update the state of class modifiers that display an icon.
selectAll() Selects all of the input's content.
setHelperTextContent(content: string) Sets the content of the helper text.
Events Description
change(string | null) Emitted whenever the value of the input changes.
