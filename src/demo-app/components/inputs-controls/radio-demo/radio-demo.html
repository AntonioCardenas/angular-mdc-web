Design & API Documentation

mdc-radio

Properties Description
id: string Unique Id of the radio button (auto generated if not supplied).
name: string Name of the radio button group.
value: any Value of the radio button.
tabIndex: number Set the underlying tab index of the radio button. (Default: 0)
ariaLabel: string Used to set the 'aria-label' attribute on the underlying input element.
ariaLabelledby: string The 'aria-labelledby' attribute takes precedence as the element's text alternative.
checked: boolean Use to verify the checked value.
disabled: boolean Disables the component.
Methods Description
focus() Set focus to the radio button.
isChecked(): boolean Returns whether or not the native input control is checked.
setChecked(checked: boolean) Sets the value of native input control to checked.
isDisabled(): boolean Returns whether or not the native input control is disabled.
setDisabled(disabled: boolean) Sets the native input control to disabled.
getValue(): any Returns the value of native input control.
setValue(value: any) Sets the value of native input control.
Event Description
change Event dispatched on value change.
