Checkbox

Checkbox component is a spec-aligned checkbox component adhering to the Material Design checkbox requirements.
Component
mdc-checkbox
Properties Description
id: string Unique Id of the checkbox (auto generated if not supplied).
name: string Name of the checkbox.
value: any Value of the checkbox.
tabIndex: number Set the underlying tab index of the checkbox. (Default: 0)
ariaLabel: string Used to set the 'aria-label' attribute on the underlying input element.
ariaLabelledby: string The 'aria-labelledby' attribute takes precedence as the element's text alternative.
checked: boolean Use to verify the checked value.
indeterminate: boolean To represent a checkbox with three states (e.g. a nested list of checkable items).
disabled: boolean Disables the component.
indeterminateToChecked: boolean Whether the checkbox should go to checked state or unchecked when toggled from indeterminate state.
Methods Description
focus() Use to set focus to the checkbox.
toggle() Toggles checkbox via user action. When it is indeterminate, toggle can go to checked or unchecked, depending on state.
Events Description
change(MdcCheckBox) Event dispatched on value change.
indeterminateChange( ) Emit when checkbox goes in and out of indeterminate state, but not when set to checked.
