Buttons

The MDC Button component is a spec-aligned button component adhering to the Material Design button requirements.
Selectors
button[mdc-button]
a[mdc-button]
Properties Description
@Input() type: string Input type of button (e.g. button, submit).
@Input() dense: boolean Compresses the button text to make it slightly smaller.
@Input() raised: boolean Elevates the button and creates a colored background.
@Input() compact: boolean Reduces the amount of horizontal padding in the button.
@Input() primary: boolean Colors the button with the primary color.
@Input() accent: boolean Colors the button with the accent color.
@Input() cardAction: boolean Correctly sets position of card action buttons. (Sets compact to true automatically)
@Input() disabled: string Disables the component.
Event
@HostListener() keydown Show ripple on keyboard down event.
@HostListener() keyup Show ripple on keyboard up event.
