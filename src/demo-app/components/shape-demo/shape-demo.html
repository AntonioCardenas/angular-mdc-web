Shape

Shapes direct attention, identify components, communicate state, and express brand.

Many MDC Web components include direct support for rounded corners; this package provides additional support for applying angled corners to unelevated surfaces.

Design & API Documentation

mdc-shape-container, mdcShapeContainer

Mandatory. Parent element containing the component to be masked.

mdcShapeContainerCorner

Mandatory. Element for masking a specific corner; there may be up to 4.
Set element to an angled corner. 
Valid values: top-left | top-right | bottom-left | bottom-right
Methods Description
setCorner(corner: string) Set element to an angled corner. 
Valid values: top-left | top-right | bottom-left | bottom-right
