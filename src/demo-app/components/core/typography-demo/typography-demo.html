Typography

MDC Typography directives implement the Material Design typography guidelines.
Directive Format
mdc-typography-button
Button
mdc-typography-body1
Body 1
mdc-typography-body2
Body 2
mdc-typography-subheading1
Subheading 1
mdc-typography-subheading2
Subheading 2
mdc-typography-caption
Caption
mdc-typography-title
Title
mdc-typography-headline
Headline
mdc-typography-display1
Display 1
mdc-typography-display2
Display 2
mdc-typography-display3
Display 3
mdc-typography-display4
Display 4
mdc-typography-adjust-margin 
<body mdc-typography>
  <h1 mdc-typography-display4 mdc-typography-adjust-margin>
    Big header
  </h1>
  <p mdc-typography-body1 mdc-typography-adjust-margin>
    A paragraph with
    <span mdc-typography-body2 mdc-typography-adjust-margin>
      emphasis.
    </span>
  </p>
</body>

In order to minimize unexpected behavior, the style classes only specify font properties, such as size, weight and line height. This means that while text will be correctly styled, it may not be correctly positioned. If you include the mdc-typography--adjust-margin class, though, positioning will be adjusted according to the style
