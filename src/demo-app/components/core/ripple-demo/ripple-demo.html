Ripple

Ripple provides components (or any element) with a material "ink ripple" interaction effect.
Directive
mdc-ripple
Properties Description
ripple: MdcRipple Use ripple orchestration properties and methods.
primary: boolean Colors the ripple with the primary theme color.
secondary: boolean Colors the ripple with the secondary theme color.
disabled: boolean Disable the ripple ink.
unbounded: boolean Whether or not the ripple is unbounded (e.g.: checkbox or radio buttons).
