Toolbars

MDC Toolbar acts as a container for multiple rows containing items such as application title, navigation menu, and tabs, among other things. Toolbars scroll with content by default, but supports fixed behavior as well.
Component
mdc-toolbar
Properties Description
fixed: boolean Makes toolbar fixed on top and have persistent elevation.
waterfall: boolean Removes fixed toolbar persistent elevation while gaining elevation scrolling down the page.
flexible: boolean Apply flexible space to toolbar's first row.
flexibleDefaultBehavior: boolean Toolbar text has flexible movement.
fixedLastrow: boolean Makes only last row of fixed toolbar anchored on top.
adjustBodyMargin: boolean If true and using a fixed toolbar, a margin-top is set to prevent toolbar overlaying content. (Default is true)
fixedAdjustElement: ElementRef Reference to the toolbar's body element. Margins will be adjusted accordingly by adjustBodyMargin. Default is document.body
Event Description
change(flexibleExpansionRatio: number)
Directive
mdc-toolbar-section Toolbar sections are aligned to the toolbar's center
Properties Description
alignStart: boolean Makes section aligns to the start.
alignEnd: boolean Makes section aligns to the end.
shrinkToFit: boolean Makes section takes the width of its content.
Directive
mdc-toolbar-row MDC Toolbars can accommodate multiple rows.
mdc-toolbar-title Styles toolbar text representing a page's title, or an application name.
mdc-toolbar-fixed-adjust Optional directive to add a default margin-top to mdc-toolbar's adjacent sibling element.
mdc-toolbar-menu-icon Left most icon of a mdc-toolbar.
mdc-toolbar-icon Icons placed on the right side of a mdc-toolbar.
