Icon

A spec-aligned material icon.
Directive
mdc-icon
Properties Description
fontSet: string Font set that the icon is a part of.
fontIcon: string Name of an icon within a font set.
fontSize: number 
The size of the icon.
If no size is specified, the default of 24px is used.
Using Material Icons home 
<mdc-icon>home</mdc-icon>
home 
<mdc-icon [fontSize]="48">home</mdc-icon>
Using Font Awesome 
<mdc-icon fontSet="fa" fontIcon="fa-camera"></mdc-icon>
Using Font Awesome w/size modifier 
<mdc-icon fontSet="fa" fontIcon="fa-map fa-4x"></mdc-icon>
