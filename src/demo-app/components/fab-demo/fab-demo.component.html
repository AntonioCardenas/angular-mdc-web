Floating Action Buttons

The MDC FAB component is a spec-aligned button component adhering to the Material Design button requirements.
Component
button[mdc-fab]
a[mdc-fab]
Properties Description
@Input() mini: boolean Makes the fab smaller (40 x 40 pixels).
@Input() plain: boolean Makes the FAB have a white background.
@Input() disabled: boolean Disables the button.
@Input() disableRipple: boolean
Child Directive
mdc-fab-icon 
<button mdc-fab
   [mini]="isMini"
   [plain]="isPlain"
   [disabled]="isDisabled">
  <mdc-fab-icon material-icon>edit</mdc-fab-icon>
</button>
