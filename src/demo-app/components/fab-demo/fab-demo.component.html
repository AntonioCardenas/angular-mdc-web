Floating Action Buttons

The MDC FAB component is a spec-aligned button component adhering to the Material Design button requirements.
Selectors
button[mdc-fab]
a[mdc-fab]
Properties Description
@Input() mini: boolean Makes the fab smaller (40 x 40 pixels).
@Input() plain: boolean Makes the FAB have a white background.
@Input() disabled: string Disables the component.
Event Description
@HostListener() keydown Show ripple on keyboard down event.
@HostListener() keyup Show ripple on keyboard up event.

Try it out

<button mdc-fab>
  <mdc-fab-icon>person</mdc-fab-icon>
</button>

Mini

<button mdc-fab [mini]="true">
  <mdc-fab-icon>search</mdc-fab-icon>
</button>

Plain

<button mdc-fab [plain]="true">
  <mdc-fab-icon>person</mdc-fab-icon>
</button>

Plain and Mini

<button mdc-fab [plain]="true" [mini]="true">
  <mdc-fab-icon>search</mdc-fab-icon>
</button>

Disabled

<button mdc-fab [disabled]="true">
  <mdc-fab-icon>person</mdc-fab-icon>
</button>
