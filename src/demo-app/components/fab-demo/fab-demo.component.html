Floating Action Button

The MDC FAB component is a spec-aligned button component adhering to the Material Design button requirements.
Component
button[mdc-fab]
a[mdc-fab]
Properties Description
mini: boolean Make the fab smaller (40 x 40 pixels).
tabIndex: number Set the tabindex of the button. (Default: 0).
disableRipple: boolean
Child Directive
mdc-fab-icon 
<button mdc-fab
   [mini]="false"
   [disableRipple]="false">
  <mdc-fab-icon material-icon>edit</mdc-fab-icon>
</button>
Using Font Awesome 
<!--Sample styling-->
.fa-icon
  justify-content: center;
  font-size: 24px;

<button mdc-fab class="fa-icon">
  <mdc-fab-icon class="fa fa-keyboard-o"></mdc-fab-icon>
</button>
