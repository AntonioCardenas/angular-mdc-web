Snackbar

The MDC Snackbar component is a spec-aligned snackbar component adhering to the Material Design snackbars requirements.
Selectors
mdc-snackbar
Properties Description
@Input() alignStart: boolean Set snackbar message as start-aligned.
@Input() dismissOnAction: boolean If you want the snackbar to remain visible until the timeout is reached (regardless of whether the user pressed the action button or not) you can set the dismissesOnAction property to false.
Methods Parameters
show(data: SnackbarMessage)
