Lists

MDC List provides styles which implement Material Design Lists - "A single continuous column of tessellated subdivisions of equal width." Both single-line and two-line lists are supported (with three-line lists planned). MDC Lists are designed to be accessible and RTL aware.

Design & API Documentation

mdc-list-group, mdcListGroup

Wraps one or more mdc-list and/or mdc-list-divider elements.

mdcListGroupSubheader, mdc-list-group-subheader

Heading text displayed above each list in a group.

mdc-list

Properties Description
multiple: boolean Multiple items can be selected at the same time.
dense: boolean Increases the density of the list, making it appear more compact.
lines: number (Set to 2 for two line list. Default is 1) Increases the height of the row to give it greater visual separation from adjacent rows.
avatar: boolean Configures the leading tiles of each row to display images instead of icons.
border: boolean Sets a border around each list item.
interactive: boolean List items receive styles for hover, focus, and press states (including the ripple). Default is true.
Methods Parameters
setMultiple(multiple: boolean) Multiple items can be selected at the same time.
setInteractive(interactive: boolean) List items receive styles for hover, focus, and press states (including the ripple).
clearSelected(skip?: MdcListItem) Clear all selected items, and optionally skip mdc-item-item.
Event Description
selectionChange: MdcListItemChange Change event that is being fired whenever the selected state of an option changes.

mdc-list-item

a[mdc-list-item]

Properties Description
selected: boolean Whether or not the item is selected. Setting this sets the selected item.
Methods Parameters
setSelected(selected: boolean) Set the selection state of the mdc-list-item.

mdcListItemGraphic

The first tile in the row (in LTR languages, the first column of the list item). Typically an icon or image.

mdcListItemMeta

The last tile in the row (in LTR languages, the last column of the list item). Typically small text, icon. or image.

mdcListItemText, mdc-list-item-text

Primary text for the row (displayed as the middle tile)

mdcListItemSecondary, mdc-list-item-secondary

Secondary text for the list item. Displayed below the primary text.

mdc-list-divider

Properties Description
inset: boolean Increases the leading margin of the divider so that it does not intersect the avatar column.
padded: boolean To make a divider match the padding of list items.
