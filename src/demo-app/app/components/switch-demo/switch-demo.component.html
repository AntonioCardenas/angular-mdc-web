Switch

The MDC Switch component is a spec-aligned switch component adhering to the Material Design switch requirements.

Google Material Components documentation

Usage

API reference

Attribute Description 
@Input() disabled: boolean
 Disables the component.
Event Description 
@Output() change
 Event dispatched on value change.

Examples

<mdc-form-field>
  <mdc-switch [(ngModel)]="isSwitchOn" [checked]="isChecked"></mdc-switch>
  <label>Label text</label>
</mdc-form-field>

Disabled

<mdc-form-field>
  <mdc-switch [disabled]="true"></mdc-switch>
  <label>Label text</label>
</mdc-form-field>
