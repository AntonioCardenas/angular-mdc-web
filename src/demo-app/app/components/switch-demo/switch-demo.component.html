Switch

The MDC Switch component is a spec-aligned switch component adhering to the Material Design switch requirements.

Google's MDC Documentation

Usage

API reference

Attribute 
@Input() id: string
 Unique Id of the switch button (auto generated if not supplied). 
@Input() ariaLabel: string
 Used to set the 'aria-label' attribute on the underlying input element. 
@Input() ariaLabelledby: string
 The 'aria-labelledby' attribute takes precedence as the element's text alternative. 
@Input() checked: boolean
 Use to verify the checked value. 
@Input() disabled: boolean
 Disables the component.
Event 
@Output() change
 Event dispatched on value change.

Examples

<mdc-form-field>
  <mdc-switch [(ngModel)]="isSwitchOn"></mdc-switch>
  <label>Label text</label>
</mdc-form-field>

Disabled

<mdc-form-field>
  <mdc-switch [disabled]="true"></mdc-switch>
  <label>Label text</label>
</mdc-form-field>
