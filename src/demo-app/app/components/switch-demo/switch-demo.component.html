Switch

The MDC Switch component is a spec-aligned switch component adhering to the Material Design switch requirements.

Google Material Components documentation

Usage

Directives

API reference

Attribute Description 
@Input() checked: boolean
 Boolean for verifying the checked value. 
@Input() labelId: string
 
@Input() disabled: boolean
 Disables the component.

Events

Event Description 
@Output() change
 Event dispatched on value change.

Examples

<mdc-form-field>
  <mdc-switch [(ngModel)]="isSwitchOn"></mdc-switch>
  <label mdc-switch-label [id]="labelId">Label text</label>
</mdc-form-field>

Disabled

<mdc-form-field>
  <mdc-switch [disabled]="true"></mdc-switch>
  <label mdc-switch-label>Label text</label>
</mdc-form-field>
