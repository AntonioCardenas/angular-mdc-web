Menu

The MDC Menu component is a spec-aligned menu component adhering to the Material Design menu requirements.

Google Material Components documentation

Usage

Directives

API reference

Attribute Description 
@Input() items: MenuItemDirective[]

Events

Event Description 
cancel
 Dispatches an event that the menu closed with no selection made. 
select
 Dispatches an event that a menu item has been selected.

Examples

<div class="mdc-menu-anchor">
  <button mdc-fab aria-label="Profile" (click)="showMenu();">
    <span mdc-fab-icon>person</span>
  </button>
  <mdc-menu #menu (cancel)="handleMenuCancel();"
   (select)="handleMenuSelect($event);"
   [items]="menuItems"></mdc-menu>
</div>
