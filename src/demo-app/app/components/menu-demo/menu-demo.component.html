Menu

The MDC Menu component is a spec-aligned menu component adhering to the Material Design menu requirements.

Google Material Components documentation

Usage

Directives

mdc-menu-anchor
mdc-menu-item
mdc-menu-divider

API reference

Method 
open(focusIndex: number)
 Opens the menu. Takes an focusIndex property that specifies the index of the menu item to be focused. If the focusIndex is omitted, no menu item will be focused. 
isOpen()
 Returns whether or not the menu is open. 
close()
 Closes the menu.
Event 
cancel
 Dispatches an event that the menu closed with no selection made. 
select
 Dispatches an event that a menu item has been selected.
