Cards

The MDC Card component is a sheet of material that serves as an entry point to more detailed information.

API reference

Attribute 
@Input() darkTheme: boolean
 Apply dark theme directly to the card. 
@Input() large: boolean
 An option to make the title text larger. (default is true) 
@Input() vertical: boolean
 Option to lay actions out vertically instead of horizontally. 
@Input() size: number
 Some predefined media item sizes. Valid values are [1, 1.5, 2, and 3]
