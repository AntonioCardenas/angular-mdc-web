Toolbars

The MDC Toolbar component is a spec-aligned button component adhering to the Material Design button requirements.

Google's Material Components for the web documentation

Usage

Components

Directives

API reference

Attribute Description 
@Input() fixed: boolean
 Makes toolbar fixed on top and have persistent elavation. 
@Input() waterfall: boolean
 Removes fixed toolbar persistent elavation; gaining elevation scrolling down the page. 
@Input() flexible: boolean
 Makes toolbar first row has flexible space. 
@Input() flexibleTitle: boolean
 Toolbar text has flexible movement. 
@Input() fixedLastrow: boolean
 Makes only last row of fixed toolbar anchored on top.

Events

Event Description 
@Output() change
Emits the ratio of current flexible space to total flexible space height.

Examples

menu Title file_download print bookmark 
<mdc-toolbar 
  [flexible]="false"
  [fixed]="false"
  [waterfall]="false"
  [fixedLastrow]="false"
  [flexibleTitle]="true">
  <mdc-toolbar-row>
    <mdc-toolbar-section [alignStart]="true">
      <i class="material-icons">menu</i>
      <span mdc-toolbar-title>Title</span>
    </mdc-toolbar-section>
    <mdc-toolbar-section [alignEnd]="true">
      <i href="#" class="material-icons" aria-label="Download" alt="Download">file_download</i>
      <i href="#" class="material-icons" aria-label="Print this page" alt="Print this page">print</i>
      <i href="#" class="material-icons" aria-label="Bookmark this page" alt="Bookmark this page">bookmark</i>
    </mdc-toolbar-section>
  </mdc-toolbar-row>
</mdc-toolbar>>
