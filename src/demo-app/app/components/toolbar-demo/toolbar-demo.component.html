Toolbars

MDC Toolbar acts as a container for multiple rows containing items such as application title, navigation menu, and tabs, among other things. Toolbars scroll with content by default, but supports fixed behavior as well.

Google's MDC Documentation

Usage

Directives

mdc-toolbar

Input fixed: boolean; // Makes toolbar fixed on top and have persistent elevation.
Input waterfall: boolean; // Removes fixed toolbar persistent elevation while gaining elevation scrolling down the page.
Input flexible: boolean; // Apply flexible space to toolbar's first row.
Input flexibleDefaultBehavior: boolean; // Toolbar text has flexible movement.
Input fixedLastrow: boolean; // Makes only last row of fixed toolbar anchored on top.

mdc-toolbar-row

mdc-toolbar-section

Input alignStart: boolean
Input alignEnd: boolean
Input shrinkToFit: boolean

mdc-toolbar-title

mdc-toolbar-fixed-adjust

mdc-toolbar-icon

mdc-toolbar-icon-menu
